# Ob-havo Ilovasi (Weather App)

Bu loyiha ikkita qismdan iborat: **Backend** (Node.js) va **Frontend** (React).

## Qanday qilib ishga tushiriladi?

Loyihani ishga tushirish uchun ikkita terminal ochishingiz kerak.

### 1-qadam: Backend-ni ishga tushirish
1. Terminalda `weather-app/backend` papkasiga kiring:
   ```bash
   cd weather-app/backend
   ```
2. Loyihani ishga tushiring:
   ```bash
   npm run dev
   ```
   *Server 3001-portda ishga tushadi.*

### 2-qadam: Frontend-ni ishga tushirish
1. Ikkinchi terminalda `weather-app/frontend` papkasiga kiring:
   ```bash
   cd weather-app/frontend
   ```
2. Loyihani ishga tushiring:
   ```bash
   npm run dev
   ```
   *Vite sizga URL beradi (odatda http://localhost:5173).*

### 3-qadam: Bravuzerda ochish
Vite bergan URL-ni brauzeringizda oching va davlat nomini yozib qidiring!

---
**Eslatma:** Agar birinchi marta ishlatayotgan bo'lsangiz va `node_modules` bo'lmasa, har bir papkada `npm install` buyrug'ini yozishingiz kerak bo'ladi.
